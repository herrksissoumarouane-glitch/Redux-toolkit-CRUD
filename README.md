# Redux Toolkit CRUD App

Eine moderne CRUD-Anwendung mit React und Redux Toolkit.

## Funktionen

- Elemente hinzufügen
- Elemente bearbeiten
- Elemente löschen
- Globale State-Verwaltung mit Redux Toolkit
- Responsive Benutzeroberfläche

## Verwendete Technologien

- React
- Redux Toolkit
- JavaScript
- CSS

## Installation

```bash
git clone https://github.com/herrksissoumarouane-glitch/Redux-toolkit-CRUD.git
cd Redux-toolkit-CRUD
npm install
npm start
````

## Projektstruktur

```bash
src/
├── components/
├── redux/
├── pages/
└── App.js
```
